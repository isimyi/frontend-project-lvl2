### Installation
1. Clone project 
    ```
    git@github.com:isimyi/frontend-project-lvl2.git
    ```
2. Go to project directory
    ```
    cd frontend-project-lvl2
    ```
3. Install dependencies
    ```
    make install
    ```
4. Create symlink to the package in global folder
    ```
    npm link
    ```
